# Computer Graphics

A collection of  projects and lab assignments demonstrating fundamental 2D graphics concepts, drawing algorithms, and transformations using C++ and OpenGL.

---

## 📚 Topics Covered

* OpenGL Environment Setup
* Basic Shape Drawing
* East West University Logo Drawing
* Village Scene Creation Using 2D Graphics
* DDA (Digital Differential Analyzer) Algorithm
* Bresenham’s Line Drawing Algorithm
* 2D Transformation
* 2D Transformation and Scaling

---

## 🛠 Technologies Used

* C++
* OpenGL
* GLUT / FreeGLUT

---

## 📂 Project List

### 1. OpenGL Environment Setup

Installation and configuration of the OpenGL development environment for graphics programming.

### 2. Basic Shape Drawing

Implementation of basic geometric shape drawing using OpenGL primitives.

### 3. Draw EWU

Creation of the East West University logo using 2D graphics techniques.

### 4. Simple Village Scene Using 2D Graphics

A simple graphical village scene developed using OpenGL and 2D rendering concepts.

### 5. DDA Line Drawing Algorithm

Implementation of the Digital Differential Analyzer (DDA) line drawing algorithm.

### 6. Bresenham’s Line Drawing Algorithm

Implementation of Bresenham’s efficient line drawing algorithm for raster graphics.

### 7. 2D Transformation

Implementation of basic 2D transformations including translation, rotation, and reflection.

### 8. 2D Transformation and Scaling

Implementation of scaling operations along with other 2D transformation techniques.
